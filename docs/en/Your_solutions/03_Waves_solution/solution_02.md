# String Harmonics Solution

## Given:
- String length: **L = 64 cm = 0.64 m**
- Fundamental frequency: **f = 330 Hz**

## Step 1: Find the wavelength

For the fundamental harmonic, the string length is half of the wavelength:

\[
L = \frac{\lambda}{2}
\]

So,

\[
\lambda = 2L
\]

\[
\lambda = 2(0.64)
\]

\[
\lambda = 1.28\ m
\]

## Step 2: Find the wave speed

Wave speed formula:

\[
v = f\lambda
\]

Substitute the values:

\[
v = 330 \times 1.28
\]

\[
v = 422.4\ m/s
\]

## Final Answer

**The speed of the wave on the guitar string is:**

\[
\mathbf{422.4\ m/s}
\]
